# ?? Inventory Management System

A complete web-based system for managing inventory with **QR code scanning**, **borrowing tracking**, and **admin dashboard**. Perfect for schools, offices, or any organization.

---

## ? Quick Setup (2 Minutes)

```bash
# 1. Install dependencies
cd inventory/server
npm install

# 2. Create .env file (see SETUP.md for details)
# 3. Start the server
npm start
```

Visit: **http://localhost:5000/inventory**

---

## ? Features

- ?? **QR Code Scanner** - Scan items with phone camera
- ?? **Borrowing Form** - Track who borrows what and when
- ?? **Admin Dashboard** - Manage inventory & records
- ?? **Secure Login** - JWT authentication
- ?? **Mobile Ready** - Works on all devices
- ? **Real-time Updates** - Live notifications
- ?? **Public Access** - Deploy online with Replit (FREE!)
